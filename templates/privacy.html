Privacy for beta testers

This beta analyzes lyrics and beat uploads on the Flask server hosting the app. It does not require an external AI API key and the included analysis engines run locally in the server process.

What may be processed

Text pasted into the editor, imported lyric files, uploaded beat/audio files, browser requests, and optional feedback form submissions may be processed to generate suggestions and debug the beta.

Storage

Drafts are autosaved in the tester’s browser local storage. Uploaded beats are stored in the configured runtime upload folder while the server keeps their beat-analysis state. Feedback is stored server-side as JSONL.

Raw corpus and benchmark privacy

This publish build supports compiled corpus and comparison profiles, so the public deployment does not need to include the raw RTF lyric corpus or raw commercial reference lyrics. Do not commit private lyric files or unlicensed reference corpora to a public repository unless you intend to publish them and have the rights to do so.

Beta limitations

This app is for writing assistance and beta testing. It is not a copyright registration system, legal review system, label-submission system, or permanent backup for creative work.
